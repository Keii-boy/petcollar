🐾 Pet Guardian

The next-generation smart collar that tells you "Why?"

Current pet trackers only show a dot on a map. Pet Guardian is an intelligent IoT security system that integrates GPS, a camera, and audio AI. Using "Deep Sleep" technology, it saves battery by waking only when it detects loud barking or a geofence breach, instantly sending a photo and alert to the owner.

We provide context, not just coordinates.

🌟 Features

Smart Safe Zones (Geofencing): Draw a virtual fence around your home. Get alerted if your pet leaves.

Bark-Triggered Camera: The collar listens for distress (loud barking), wakes up, takes a photo, and uploads it via 4G.

Ultra-Low Power: Over 90% of the time, the collar is in deep sleep, solving the battery drain issue of standard pet cameras.

Multi-Language Web Dashboard: Monitor all your pets from a responsive dashboard (Supports English, Japanese, Myanmar).

🏗️ System Architecture

1. Hardware (Firmware)

Board: LILYGO T-SIMCAM (ESP32 + SIM7000G)

Camera: OV2640 Module

Sensor: KY-037 Sound Sensor

Language: C++ (Arduino Framework)

2. Software (Frontend App)

Tech Stack: HTML5, Tailwind CSS, Vanilla JavaScript

Mapping: Leaflet.js

Features: Responsive design, simulated IoT events, localization.
